# DIGISOAL Website

This is the codebase for the official DIGISOAL single-page website, built according to the provided project brief. It is a mobile-first, responsive site with a focus on fluid, shapeshifting transitions and a modern, professional aesthetic.

## Features

- **Single-Window Navigation:** The site is designed for a continuous scrolling experience, with touch-first navigation.
- **Fluid Morphing Transitions:** Animations are implemented using GSAP and its plugins to create shapeshifting effects between sections.
- **Translucent UI:** Utilizes CSS `backdrop-filter` for a "glass" effect on UI elements.
- **Responsive & Accessible:** Built with a mobile-first approach, ensuring a great experience across all devices, with ARIA attributes and keyboard navigation support.
- **SEO-Optimized:** Includes semantic HTML5, meta tags, and Schema.org markup.
